Lecture 5: CSS Box Model & Layout

Summary

The CSS Box Model describes how the elements on a web page are structured and spaced. It includes content, padding, border, and margin. Understanding the box model helps in designing layouts and spacing elements properly.

Content

Topic Description
CSS Box Model Every HTML element is a rectangular box with content, padding, border, and margin.
Padding & Margin Padding adds space inside the element. Margin adds space outside the element.
CSS Layout Layouts can be created using display properties like block, inline, flex, and grid.

Example

Example Code

div{
width: 200px;
padding: 20px;
border: 2px solid blue;
margin: 10px;
}
