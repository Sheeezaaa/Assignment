Lecture 6: Responsive Web Design

Summary

Responsive Web Design (RWD) allows websites to adapt to different screen sizes and devices like mobiles, tablets, and desktops. It improves user experience by making the site readable and navigable on all devices.

Content

Topic Description
Media Queries Media queries allow applying different CSS styles based on device screen width or type.
Flexible Layouts Using percentages and relative units instead of fixed pixels to make layouts flexible.
Responsive Images Images should resize according to the device screen to fit properly without breaking the layout.

Example

Example Code

@media screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
  div.container {
    width: 100%;
    padding: 10px;
  }
}
