Lecture 4: Introduction to CSS

Summary

CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, spacing, and layout of HTML elements. CSS helps make websites more attractive and visually appealing.

Content

Topic Description
What is CSS CSS is used to style HTML elements and control the appearance of web pages.
Types of CSS There are three types of CSS: Inline CSS, Internal CSS, and External CSS.
CSS Selectors Selectors are used to target HTML elements that we want to style.

Example

Example Code

h1{
color: blue;
text-align: center;
}

p{
color: gray;
font-size: 16px;
}
